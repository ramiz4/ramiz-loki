### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/ramiz4/ramiz-loki.git
   cd ramiz-loki
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`
